    The BSON specification allows you to set a subType for every Binary. It says: subtype ::= "\x00" Generic binary subtype | "\x01" Function | "\x02" Binary (Old) | "\x03" UUID (Old) | "\x04" UUID | "\x05" MD5 | "\x80" User defined
